Google Search & Screenshot Automation Bot

This project is a simple yet powerful web automation script built with Python, Selenium, and Undetected ChromeDriver.  
It automatically searches "House of Dragon" on Google, opens a result page, navigates to an image section, and captures a screenshot

=> Features

-> Opens Google and searches for a keyword.
-> Clicks on a search result and navigates to the target page.
-> Opens an image section related to the keyword.
-> Takes a screenshot and saves it locally.
-> Uses Undetected ChromeDriver to avoid being blocked.

=>Tech Stack

->Python 3.8+
->Selenium
->Undetected-Chromedriver
->ChromeDriver
